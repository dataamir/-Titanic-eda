# 🚢 Titanic — Exploratory Data Analysis

## 📌 Project Overview
Exploratory Data Analysis (EDA) on the classic Titanic dataset
to uncover patterns in passenger survival.

## 📊 Key Findings
- Women had a **83% survival rate** vs men at **17%**
- 1st class passengers survived significantly more than 3rd class
- Higher fare correlated with better survival chances

## 🛠️ Tools & Libraries Used
